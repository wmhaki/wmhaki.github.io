# Privacy Policy — SPMP Mark Keyin

This repository hosts the privacy policy page for the **SPMP Mark Keyin** Chrome extension, published via GitHub Pages.

🔗 **Live page:** https://wmhaki.github.io

## About

SPMP Mark Keyin is a Chrome extension that helps lecturers transfer student marks from Excel/CSV into the SPMP grade-entry system. This privacy policy explains what data the extension handles and confirms no data is ever sent to a server.
